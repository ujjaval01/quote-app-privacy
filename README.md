# 📖 Daily Quotes - Android App

Daily Quotes is a modern Android application built with Jetpack Compose that delivers motivational and inspirational quotes every day.

---

## ✨ Features

- 🎯 Daily motivational quote slider
- 📚 Category-based quote browsing
- ❤️ Save favorite quotes locally
- 🔔 Daily quote notification
- 📤 Share quotes as text
- 🖼 Save quotes as image
- 🌙 Dark & Light mode support
- ⚡ Built entirely using Jetpack Compose

---

## 🛠 Built With

- Kotlin
- Jetpack Compose
- Material 3
- MVVM Architecture
- Local JSON Data Storage

---

## 🔒 Privacy
